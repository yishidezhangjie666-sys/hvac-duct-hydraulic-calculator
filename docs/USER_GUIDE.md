# 用户使用指南

## 适用对象

本工具适合建筑环境与能源应用工程、暖通、通风、空调水系统相关课程设计、工程初步核算和个人作品集展示。

本工具不替代正式工程设计、设备样本选型或规范校核。

## 快速开始

1. 打开在线地址或本地运行项目；
2. 在侧边栏选择计算模块；
3. 点击加载示例数据，熟悉输入字段；
4. 按项目实际参数修改表格；
5. 查看计算结果、系统汇总和校核提示；
6. 根据需要导出 CSV、Excel 或 Word 计算说明书。

## 模块选择建议

| 需求 | 建议模块 |
|---|---|
| 计算风管风速、阻力、动压 | 通风风管水力计算 |
| 计算空调水系统流量、管径、阻力和水泵推荐 | 空调水系统水力计算 |
| 初步校核风机盘管和新风机组 | 空调末端设备初步选型 |
| 初步校核风冷热泵、冷水机组和锅炉 | 冷热源设备初步选型 |
| 初步校核风机和水泵拟选参数 | 风机 / 水泵选型校核 |

## 输入数据建议

- 优先使用统一单位；
- 不确定的参数先参考课程设计手册或设备样本；
- 风量、流量、负荷、扬程、风压等关键参数不要留空；
- 对零值、负值和异常效率保持警惕；
- 出现“需复核”时，优先检查输入数据。

## 结果解读

工具中的“合理、偏大、不足、明显偏大、需复核”等状态均为简化校核提示。

状态提示用于帮助用户发现可能不合理的输入或选型，不等同于正式规范判断。

## 导出建议

- CSV 适合继续整理数据；
- Excel 适合查看表格和汇总；
- Word 计算说明书适合学习、课程设计辅助核算和过程展示；
- 正式课程设计或工程文件仍需按老师或单位要求重新整理。

## 常见问题

### 为什么会出现需复核？

通常是因为输入缺失、零值、负值、效率异常或工况边界不明确。

### Word 导出能不能直接作为正式课程设计？

不能。当前 Word 导出是普通计算说明书，不套正式课程设计排版规则。

### 计算结果能不能直接用于工程？

不能。实际工程应结合现行规范、设计手册、设备样本、厂家曲线和工程经验复核。
