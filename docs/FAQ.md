# 常见问题

## 这个工具能替代正式工程设计吗？

不能。本工具采用简化工程计算口径，主要用于学习、课程设计辅助核算和工程初步校核。实际工程设计应结合现行规范、设计手册、设备样本、厂家曲线和工程经验复核。

## 这个工具适合谁用？

适合建环专业学生、课程设计辅助核算、暖通计算流程学习、个人作品集展示，以及工程初步校核参考。它更适合帮助理解流程和整理结果，不适合作为最终设计结论的唯一依据。

## 为什么要做 Word 导出？

Word 导出可以保存计算过程、输入参数、计算结果、公式说明和免责声明，便于学习记录、课程设计辅助整理和后续复核。相比只看页面结果，导出文档更适合归档和说明。

## 为什么有些结果会显示“需复核”？

通常是因为输入缺失、零值、负值、效率异常，或者工况边界不够明确。出现“需复核”时，应先检查输入参数、单位和工况含义，不要直接采用结果。

## 为什么不接入厂家设备库？

厂家样本、性能曲线和实际选型约束比较复杂。当前版本暂不引入厂家库，是为了避免把简化校核工具包装成正式选型软件，也避免给出超出当前项目边界的结论。

## 为什么不做 AI 自动生成计算书？

当前项目优先保证计算流程、导出结构、测试和工程边界稳定。AI 自动生成会增加不可控内容，可能影响计算说明的一致性和可复核性，所以暂不纳入当前版本。

## 线上工具和本地运行有什么区别？

线上工具适合快速体验和展示；本地运行适合开发、测试和二次修改。本地运行需要安装 Python 依赖，并按 README 中的步骤启动 Streamlit。

## 后续会继续扩展吗？

会优先考虑 README 展示优化、截图更新、示例数据扩充和 Word 导出细节优化。大型新模块需要谨慎评估公式来源、输入边界、导出结构和测试覆盖后再推进。
