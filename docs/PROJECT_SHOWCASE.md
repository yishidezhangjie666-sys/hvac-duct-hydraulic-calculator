# 项目展示说明

## 项目定位

建环工程计算工具箱是一个面向建筑环境与能源应用工程方向的 Streamlit Web 工具项目。项目将通风风管、空调水系统、空调末端设备、冷热源设备以及风机 / 水泵选型校核等常见计算流程整合到同一个在线入口，方便学习、课程设计辅助核算、工程初步校核和个人作品展示。

项目当前稳定版本为 `v0.2.1`。它不是正式工程设计软件，也不能替代规范校核、设备样本选型、厂家性能曲线复核或注册工程师审查。

## 解决的问题

在建环专业学习和课程设计过程中，常见痛点包括：

- 计算过程分散在 Excel、手算草稿和 Word 文档中，不方便复核；
- 同一类管段或设备方案需要重复录入、重复计算；
- 结果表格、系统汇总、公式说明和导出文件不容易保持一致；
- 初学者容易只关注数值结果，而忽略风速、流速、余量、效率和异常输入的复核提示。

本项目将这些流程做成 Web 工具，提供统一的输入、计算、校核、说明和导出入口。

## 当前功能

| 模块 | 主要能力 | 输出 |
|---|---|---|
| 通风风管水力计算 | 风量换算、截面积、风速、动压、沿程阻力、局部阻力、系统总阻力 | 管段结果、系统总阻力、风机风量与风压参考 |
| 空调水系统水力计算 | 根据负荷估算水流量，计算管径、流速、阻力和水泵参数 | 管段结果、系统汇总、水泵流量与扬程推荐 |
| 空调末端设备初步选型 | 风机盘管和新风机组冷量、热量、风量余量校核 | 选型余量、状态提示、复核建议 |
| 冷热源设备初步选型 | 风冷热泵、冷水机组、锅炉 / 热源设备容量与台数校核 | 容量余量、温差校核、复核建议 |
| 风机 / 水泵选型校核 | 风机风量 / 风压 / 电机功率和水泵流量 / 扬程 / 功率校核 | 余量状态、异常输入提示、多格式导出 |

所有模块均支持 CSV、Excel 和 Word 计算说明书导出。

## 技术实现

- 应用框架：Streamlit
- 数据处理：pandas
- Excel 导出：openpyxl
- Word 导出：python-docx
- 自动化检查：compileall、pytest、GitHub Actions
- 部署平台：Streamlit Community Cloud

代码结构采用 `modules/` 和 `utils/` 分离：计算模块、导出工具和 Word 报告生成逻辑相互解耦，方便后续维护和扩展。

项目当前已完成以下质量保障：

## 项目边界

本项目采用简化工程计算口径，适合学习、课程设计辅助核算、工程初步校核和作品展示。实际工程设计仍应结合现行规范、设计手册、设备样本、厂家曲线和工程经验进行复核。

## 适合展示的能力

- 将专业计算流程产品化为可在线访问的 Web 工具；
- 将输入、计算、结果解释和导出流程整合到同一应用；
- 在持续迭代中保持测试、文档和发布记录；
- 对计算边界、适用场景和结果风险保持明确说明。
