## 项目简介

建环工程计算工具箱是一个面向建筑环境与能源应用工程专业学习、课程设计辅助核算和工程初步校核的在线工具。它把建环 / 暖通常见计算流程整理成 Streamlit Web 模块，让用户可以在一个入口中完成参数录入、示例数据加载和结果查看。

项目支持结果校核、结果解释以及 CSV / Excel / Word 计算说明书导出，便于学习记录、课程设计辅助整理和作品集展示。当前版本采用简化工程计算口径，不能替代正式工程设计。

## 功能模块

1. 通风风管水力计算：用于矩形风管管段的风量换算、风速、动压、阻力和系统总阻力初步计算。
2. 空调水系统水力计算：用于根据负荷和温差估算水流量，并进行管径、流速、阻力和水泵参数初步校核。
3. 空调末端设备初步选型：用于风机盘管和新风机组的冷量、热量、风量余量校核。
4. 冷热源设备初步选型：用于风冷热泵、冷水机组、锅炉 / 热源设备的容量、台数和温差初步校核。
5. 风机 / 水泵选型校核：用于风机风量 / 风压 / 功率和水泵流量 / 扬程 / 功率的简化校核。
