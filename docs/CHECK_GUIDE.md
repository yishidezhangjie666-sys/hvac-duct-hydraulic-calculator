# 校核状态说明

本说明用于解释建环工程计算工具箱中常见校核状态的含义。

所有校核状态均为简化建议，用于学习、课程设计辅助核算和工程初步校核展示，不能替代正式工程设计、设备样本选型或规范校核。

## 通风风管水力计算

常见状态包括：

- 风速偏低：说明当前风量和管段尺寸组合下风速较低，可结合布置、经济性和风管尺寸复核。
- 风速偏高：说明当前风量和管段尺寸组合下风速较高，应关注噪声、阻力和管段尺寸。
- 阻力较大：通常与较长管段、较大局部阻力系数或较高风速有关。

## 空调水系统水力计算

常见状态包括：

- 流速偏低：可能说明管径偏大或流量偏小，应结合系统经济性和水力平衡复核。
- 流速合适：说明在当前简化阈值下流速处于建议范围。
- 流速偏高：可能说明管径偏小，应关注阻力、噪声和水泵扬程。

## 空调末端设备初步选型

常见状态包括：

- 建议可用：额定冷量、热量或风量在简化校核范围内。
- 拟选偏小：至少有一项能力不足，应重新核对设备型号或负荷。
- 余量偏大：设备能力明显高于需求，应关注初投资、运行调节和舒适性。
- 需复核：输入参数、焓差或工况不适合直接计算，应重新核对数据。

### 建议处理顺序

1. 先复核负荷、风量、流量等基础输入；
2. 再复核设备额定能力和台数；
3. 最后结合设备样本、运行调节、噪声、能效和工程布置判断。

## 冷热源设备初步选型

常见状态包括：

- 建议可用：总装机容量相对所需容量处于简化建议范围。
- 拟选偏小：总装机容量低于所需容量。
- 余量偏大 / 明显偏大：装机容量明显高于负荷需求，应关注初投资、部分负荷效率和运行调节。
- 容量可用，温差需复核：容量基本满足，但供回水温差不在常见简化范围内。
- 需复核：输入数据或温差条件不适合直接判断。

### 建议处理顺序

1. 先复核设计冷 / 热负荷和备用系数；
2. 再复核单台设备容量和台数；
3. 对温差需复核的方案，优先检查供回水温度设定；
4. 最后结合厂家样本、部分负荷效率、安装条件和运行策略判断。

## 使用建议

1. 先看校核状态，再回到输入表检查关键参数。
2. 对“偏大”“偏低”“偏高”的结果，不要直接视为错误，应结合工程背景复核。
3. 对“拟选偏小”“明显偏大”“需复核”的结果，应优先检查负荷、风量、流量、管径、设备容量和工况参数。
4. 本工具只提供简化初步判断，不替代正式设计。
