# v0.2.1 Roadmap

## 版本定位

`v0.2.1` 是 `v0.2.0` 发布后的维护型小版本，目标是优化文档一致性、Word 导出可读性、使用说明和线上体验细节。

本版本不计划新增大型计算模块，不调整核心计算公式，不修改校核阈值。

## 计划任务

### 1. Word 计算说明书可读性优化

- [x] 优化 Word 报告标题、章节、表格、说明与免责声明的排版一致性
- [x] 改善宽表格在 Word 中的可读性
- [x] 增强 Word 导出结构测试
- [x] 保持普通计算说明书格式，不套正式课程设计排版规则

### 2. 文档和使用说明优化

- [x] 新增用户使用指南
- [x] 新增 Word 导出说明
- [x] README 增加文档入口
- [x] 检查 README 中截图、链接和版本说明一致性

### 3. 线上体验小修

- [x] 侧边栏显示当前稳定版本
- [x] 侧边栏提供说明文档入口
- [x] 保持五个模块入口顺序稳定
- [x] 清理结果表空值显示，避免页面直接出现 None、NaN 或 <NA>
- [x] 线上页面回归检查

### 4. 测试增强

- [x] 增加文档链接测试
- [x] 增加 Word 导出可读性测试
- [x] 增加版本边界测试
- [x] 保持现有计算测试通过

## 暂不计划

- 不新增用户登录
- 不接入数据库
- 不接入 AI 自动生成计算书
- 不新增大型工程计算模块
- 不修改核心计算公式
- 不修改校核阈值
- 不移动或重发 v0.2.0 tag / Release

## 版本边界

本项目仍定位为学习、课程设计辅助核算、工程初步校核和个人作品集展示工具。

计算结果不能替代正式工程设计、设备样本选型或规范校核。
