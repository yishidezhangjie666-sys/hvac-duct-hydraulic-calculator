# 示例数据说明

本项目示例数据用于展示四个模块的输入方式、计算流程和典型校核结果，适合学习、课程设计辅助核算、工程初步校核和作品集展示。

示例数据不是正式工程设计参数，不能替代规范校核、厂家样本选型或施工图设计。

## 通风风管水力计算

`sample_data.csv` 提供多个矩形风管管段，覆盖不同风量、风管尺寸、管段长度和局部阻力系数。

示例重点：

- 常规低速管段，用于展示基础水力计算过程；
- 局部阻力较大的管段，用于观察局部阻力对总阻力的影响；
- 风速偏低和偏高的管段，用于理解风速校核提示。

## 空调水系统水力计算

模块内置示例覆盖小负荷支管、普通支管、主管和阻力较大的管段。

示例重点：

- 水流量可由冷量和温差自动估算；
- 不同管径会产生不同流速校核状态；
- 示例包含偏低、合适、偏高等流速结果，便于理解管径选择对水力计算的影响。

## 空调末端设备初步选型

风机盘管和新风机组示例用于展示冷量、热量、风量和焓差的简化选型校核。

示例重点：

- 风机盘管包含建议可用、拟选偏小和余量偏大等典型结果；
- 新风机组包含建议可用、拟选偏小、余量偏大和焓差需复核等典型结果；
- 结果仅用于初步判断，实际工程还应结合设备样本、噪声、余压、水阻和控制方式复核。

## 冷热源设备初步选型

风冷热泵、冷水机组和锅炉 / 热源设备示例用于展示容量余量、台数和供回水温差的初步校核。

示例重点：

- 覆盖建议可用、拟选偏小、明显偏大和温差需复核等典型状态；
- 展示备用系数和单台容量对总装机容量的影响；
- 结果不能替代正式设备样本选型、规范校核或经济技术比较。

## 使用建议

1. 先加载示例数据，观察结果表中的校核状态。
2. 修改一个输入参数，观察风速、流速、容量余量或温差校核如何变化。
3. 导出 CSV、Excel 或 Word 计算说明书，用于学习记录和初步成果展示。
