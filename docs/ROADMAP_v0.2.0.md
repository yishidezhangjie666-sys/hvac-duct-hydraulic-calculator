# v0.2.0 Roadmap

## 版本目标

`v0.2.0` 计划聚焦于导出质量优化、示例数据完善和设备选型校核增强。

本阶段不追求一次性加入大量模块，而是优先提高已有四个模块的可用性、稳定性和展示质量。

## 计划任务

### 1. Word 计算说明书导出优化

- 统一四个模块的 Word 导出结构
- 增加更清晰的计算参数表
- 增加计算结果表
- 增加公式说明
- 增加免责声明
- 保持普通计算说明书格式，不套正式课程设计排版规则

### 2. 示例数据增强

- 为四个模块补充更贴近建环课程设计的示例数据
- 增加正常、偏大、不足、需复核等典型案例
- 方便用户理解校核结果

### 3. 设备选型校核增强

- 进一步完善风机盘管和新风机组余量判断
- 进一步完善冷热源设备容量余量提示
- 保持简化工程口径，不引入复杂规范库

### 4. 风机 / 水泵选型校核预研

- 评估是否新增风机选型校核
- 评估是否新增水泵选型校核
- 优先设计输入字段、输出字段和计算边界

### 5. 自动化检查增强

- 增加更多边界样例测试
- 增加导出文件生成测试
- 检查 README 链接和截图引用

## 暂不计划

- 暂不引入用户登录
- 暂不接入数据库
- 暂不接入 AI 自动生成计算书
- 暂不做正式工程设计软件
- 暂不引入复杂规范库

## 版本边界

`v0.2.0` 仍定位为学习、课程设计辅助核算、工程初步校核和个人作品集展示工具，不能替代正式工程设计、厂家样本选型或规范校核。
