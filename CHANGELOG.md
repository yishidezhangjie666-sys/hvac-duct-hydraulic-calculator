# Changelog

## Unreleased

### Added

- 增强四个模块示例数据，覆盖合理、不足、偏大和需复核等典型校核状态。
- 新增示例数据说明文档和覆盖测试，降低示例数据退化风险。

## v0.1.0 - 2026-06-28

### Added

- 完成 Streamlit 在线版建环工程计算工具箱基础框架。
- 新增通风风管水力计算模块。
- 新增空调水系统水力计算模块。
- 新增空调末端设备初步选型模块，包含风机盘管和新风机组初步选型。
- 新增冷热源设备初步选型模块，包含风冷热泵、冷水机组、锅炉 / 热源设备初步选型。
- 支持 CSV、Excel 和 Word 计算说明书导出。
- 新增项目级 `AGENTS.md`，明确 Codex 自动化维护边界。
- 新增 pytest 自动化测试。
- 新增本地检查脚本 `scripts/check_project.ps1`。
- 新增 GitHub Actions CI 自动检查。
- 新增 README 截图、CI 状态徽章和项目说明。

### Notes

- 本项目采用简化工程计算口径，主要用于学习、课程设计辅助核算、工程初步校核和个人作品集展示。
- 当前版本不能替代正式工程设计、设备样本选型或规范校核。
