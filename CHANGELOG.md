# Changelog

## Unreleased

## v0.2.0 - 2026-06-30

### Added

- 增强四个模块示例数据，覆盖合理、不足、偏大和需复核等典型校核状态。
- 新增示例数据说明文档和覆盖测试，降低示例数据退化风险。
- 新增校核状态说明文档，解释偏低、偏高、不足、偏大和需复核等状态含义。
- 新增风机 / 水泵选型校核预研文档，明确后续模块的输入输出、公式边界、实现顺序和暂不计划内容。
- 新增风机 / 水泵选型校核纯计算函数和测试，为后续 Streamlit 页面接入做准备。
- 新增风机 / 水泵选型校核 Streamlit 页面，支持示例数据、校核结果解读以及 CSV、Excel、Word 导出。

### Changed

- 统一四个模块的 Word 计算说明书导出结构。
- 新增 Word 导出基础测试，降低后续导出功能回归风险。
- 清理通风模块页面中过时的后续规划说明，避免与当前四模块状态冲突。
- 增强末端设备和冷热源设备模块的校核结果解读与复核建议。
- Word 计算说明书导出补充设备选型校核边界说明。
- 补充风机 / 水泵选型校核页面截图和 v0.2.0 发布说明。

## v0.1.0 - 2026-06-28

### Added

- 完成 Streamlit 在线版建环工程计算工具箱基础框架。
- 新增通风风管水力计算模块。
- 新增空调水系统水力计算模块。
- 新增空调末端设备初步选型模块，包含风机盘管和新风机组初步选型。
- 新增冷热源设备初步选型模块，包含风冷热泵、冷水机组、锅炉 / 热源设备初步选型。
- 支持 CSV、Excel 和 Word 计算说明书导出。
- 新增项目级 `AGENTS.md`，明确 Codex 自动化维护边界。
- 新增 pytest 自动化测试。
- 新增本地检查脚本 `scripts/check_project.ps1`。
- 新增 GitHub Actions CI 自动检查。
- 新增 README 截图、CI 状态徽章和项目说明。

### Notes

- 本项目采用简化工程计算口径，主要用于学习、课程设计辅助核算、工程初步校核和个人作品集展示。
- 当前版本不能替代正式工程设计、设备样本选型或规范校核。
