# v0.2.0 Release Notes

## 版本定位

`v0.2.0` 是建环工程计算工具箱在 `v0.1.0` 基础上的功能完善版本，重点提升导出质量、示例数据、校核说明和设备选型辅助能力。

本版本仍定位为学习、课程设计辅助核算、工程初步校核和个人作品集展示工具，不能替代正式工程设计、设备样本选型或规范校核。

## 在线体验

https://hvac-mep-calc-toolbox.streamlit.app/

## 主要更新

### 1. Word 计算说明书导出优化

- 统一五个模块的 Word 计算说明书导出结构；
- 导出内容包含模块说明、输入参数、计算结果、系统汇总、公式说明和免责声明；
- 保持普通计算说明书格式，不套正式课程设计排版规则。

### 2. 示例数据和校核说明增强

- 增强示例数据，覆盖合理、不足、偏大、明显偏大、需复核等典型状态；
- 新增示例数据说明文档；
- 新增校核状态说明文档；
- 页面中增加校核结果解读和复核建议。

### 3. 新增风机 / 水泵选型校核模块

- 支持风机风量、风压和电机功率参考校核；
- 支持水泵流量、扬程和电机功率参考校核；
- 支持示例数据；
- 支持 CSV、Excel 和 Word 导出；
- 支持校核结果解读和复核建议。

### 4. 自动化测试增强

- 增加 Word 导出测试；
- 增加示例数据覆盖测试；
- 增加风机 / 水泵计算测试；
- 增加风机 / 水泵 Word 导出测试；
- GitHub Actions 持续运行项目检查。

## 当前模块

1. 通风风管水力计算；
2. 空调水系统水力计算；
3. 空调末端设备初步选型；
4. 冷热源设备初步选型；
5. 风机 / 水泵选型校核。

## 工程边界

本项目采用简化工程计算口径。  
计算结果仅用于学习、课程设计辅助核算和工程初步校核。  
实际工程设计应结合现行规范、设计手册、设备样本、厂家曲线和工程经验进行复核。

## 后续方向

- 继续优化 Word 样式和可读性；
- 继续补充更细分的典型工况；
- 评估是否调整校核阈值；
- 增强线上展示和项目文档；
- 在保持简化工程口径的前提下继续扩展建环常用计算模块。
