# v0.2.1 Release Notes

## 版本定位

`v0.2.1` 是 `v0.2.0` 发布后的维护型小版本，重点优化文档一致性、Word 导出可读性、使用说明和线上体验细节。

本版本不新增大型计算模块，不修改核心计算公式，不修改校核阈值。

## 在线体验

https://hvac-mep-calc-toolbox.streamlit.app/

## 主要更新

### 1. 使用文档增强

- 新增用户使用指南；
- 新增 Word 计算说明书导出说明；
- README 新增使用文档入口；
- 新增 v0.2.1 Roadmap。

### 2. Word 导出可读性优化

- 优化 Word 报告标题、章节、表格和免责声明展示；
- 改善空值显示，避免 `None`、`NaN`、`<NA>` 直接出现在导出文档中；
- 增强 Word 导出可读性测试。

### 3. 线上体验小修

- 侧边栏增加当前稳定版本提示；
- 侧边栏增加工具使用边界提示；
- 保持五个模块入口顺序稳定；
- 清理风机 / 水泵结果表空值展示，避免页面直接显示 `None`、`NaN` 或 `<NA>`。

### 4. 测试增强

- 新增文档链接测试；
- 新增 Word 导出可读性测试；
- 新增版本边界测试；

## 当前模块

1. 通风风管水力计算；
2. 空调水系统水力计算；
3. 空调末端设备初步选型；
4. 冷热源设备初步选型；
5. 风机 / 水泵选型校核。

## 工程边界

本项目采用简化工程计算口径。

计算结果仅用于学习、课程设计辅助核算和工程初步校核。

实际工程设计应结合现行规范、设计手册、设备样本、厂家曲线和工程经验进行复核。
