# v0.1.0 Release Notes

## 版本定位

`v0.1.0` 是建环工程计算工具箱的第一个稳定整理版本，完成了从单一计算页面到四模块 Streamlit 工具箱的基础建设。

本版本主要面向学习、课程设计辅助核算、工程初步校核和个人作品集展示。

## 在线体验

https://hvac-mep-calc-toolbox.streamlit.app/

## 主要功能

- 通风风管水力计算
  - 风量换算
  - 截面积、风速、当量直径
  - 动压、沿程阻力、局部阻力、系统总阻力
  - 风速校核
- 空调水系统水力计算
  - 根据负荷和供回水温差估算水流量
  - 管径、流速、动压、沿程阻力、局部阻力
  - 水泵流量和扬程推荐
  - 流速校核
- 空调末端设备初步选型
  - 风机盘管冷量、热量、风量余量校核
  - 新风机组新风冷负荷、机组冷量、风量余量校核
  - 焓差异常提示
- 冷热源设备初步选型
  - 风冷热泵机组制冷量、制热量和台数初步校核
  - 冷水机组制冷量、台数和冷冻水温差初步校核
  - 锅炉 / 热源设备供热量、台数和供回水温差初步校核

## 导出能力

- CSV 导出
- Excel 导出
- Word 计算说明书导出

## 工程边界

本项目采用简化工程计算口径。  
计算结果仅用于学习、课程设计辅助核算和工程初步校核，不能替代正式工程设计、设备样本选型或规范校核。
