# 通风风管水力计算辅助工具

> HVAC Duct Hydraulic Calculator — 矩形风管简化水力计算

---

## 项目简介

一个基于 Python + Streamlit 的通风风管水力计算辅助工具。专为建筑环境与能源应用工程专业学生和从业者设计，用于矩形风管系统的简化水力计算。

适用于课程设计辅助核算、考研复试展示、工程计算思路演示等场景。

---

## 适用场景

- 建筑环境与能源应用工程专业课程设计辅助校核
- 通风空调风管系统的初步水力估算
- 暖通工程计算思路的快速演示
- 考研复试、实习申请、个人简历中的专业项目展示

---

## 功能说明

- 支持多个矩形风管管段的批量录入
- 可编辑的数据表格，支持增删行
- 自动计算：
  - 风量换算 (m³/h → m³/s)
  - 截面积
  - 风速
  - 当量直径（简化水力直径）
  - 动压
  - 沿程阻力（比摩阻 × 长度）
  - 局部阻力
  - 管段总阻力
- 系统总阻力汇总展示
- 可调空气密度参数（侧边栏）
- 示例数据一键加载
- CSV / Excel 导出
- 内嵌公式说明，方便截图展示

---

## 计算公式

| 参数 | 公式 |
|------|------|
| 风量 | Q = Q_h / 3600 (m³/s) |
| 截面积 | A = a × b (m²) |
| 风速 | v = Q / A (m/s) |
| 当量直径 | D_e = 2ab / (a + b) (m) |
| 动压 | P_d = ρv² / 2 (Pa) |
| 沿程阻力 | P_y = R_m × l (Pa) |
| 局部阻力 | P_j = ζ × P_d (Pa) |
| 管段总阻力 | P = P_y + P_j (Pa) |
| 系统总阻力 | ΣP = ΣP_i (Pa) |

## 安装方法

```bash
# 克隆仓库
git clone https://github.com/your-username/hvac-duct-hydraulic-calculator.git
cd hvac-duct-hydraulic-calculator

# 创建虚拟环境（可选但推荐）
python -m venv .venv

# Windows
.venv\Scripts\activate
# macOS / Linux
# source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

---

## 运行方法

```bash
streamlit run app.py
```

启动后在浏览器中打开 http://localhost:8501 即可使用。
